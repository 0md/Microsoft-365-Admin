系统信息

基于 SpringBoot 2.1.3 开发


此系统是一个基于 Azure API 管理 Microsoft 365 用户的程序, 支持 A1、A1P、A3、E3、E5 等订阅类型

支持多个管理账号切换

需要API权限：Directory.ReadWrite.All 以及 User.ReadWrite.All
